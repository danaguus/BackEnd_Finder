*** PRERREQUISITOS ***
 -> Instalar NodeJs

*** INSTRUCCIONES ***
1. Abrir un terminal (cmd para windows), ubicarlo en la carpeta raíz de este proyecto.
1. Ejecutar el comando "npm install --save"
2. Ejecutar el comando "node server/index.js"
3. En un navegador de Internet, ir a la dirección "localhost:3030" (o el que se configure en el index.js del server para la variable PORT)
